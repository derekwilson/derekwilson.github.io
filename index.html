PassTheParcel

PassTheParcel is a simple, quick and easy to use app to play music for "Pass The Parcel" or "Musical Chair" type games. I wrote this for a group of friends to play pass the parcel, it enables me to select the music I want and have it automatically stop but manually start, all without adverts. The source is hosted on bitbucket feel free to download or just browse the code.

Trailblazer

Trailblazer is a simple, quick and easy to use track recorder, for hiking, running cycling etc. You can record and share your tracks. The app was inspired by MyTracks - a Google app that is now deprecated. The app has been produced by remote developers working as a team. The source is hosted on bitbucket feel free to download or just browse the code.

Timestyle Fitbit

Timestyle Fitbit is my implementation of Dan Tilden's wonderful design for Pebble watches. Its inspired by that project and attempts to reproduce that design on Fitbit OS watches. The source is hosted on bitbucket feel free to download or just browse the code.

wrist-list

wrist-list is a todo list app for FitbitOS devices. It was written for a specific person who wanted to be able to use their Google Keep shopping lists on the watch, hopefully its written in such a way that others will find it useful as well. The source is hosted on bitbucket feel free to download or just browse the code.

wrist-spin

wrist-spin for Android phones and wrist-spin for Pebble is a live cricket score display for Android OS 5+ phones and Pebble watches and Fitbit OS 4 devices. Its designed to keep you up to date with your favourite team using glanceable displays. The source is hosted on bitbucket feel free to download or just browse the code.

Recommender

Recommender is an Android application to store, organize and share personal recommendations. Its designed to be used in person where you want to pass on a recommendation to a friend. There is no web service, there is no need to register you details anywhere. The source is hosted on bitbucket feel free to download or just browse the code.

SendRecommendation

SendRecommendation is an Windows 10 Universal application to send recommendations to the main Recommender application. The source is hosted on bitbucket feel free to download or just browse the code.

RamEater

RamEater is a memory stress tool for Android devices. The source is hosted on github feel free to download it or just browse the code.

MeasureMe

MeasureMe is a low friction personal measurement tool for Android devices. The source is hosted on bitbucket feel free to download it or just browse the code. There are also example spreadsheets available to download

Podcast Utilities

Podcast Utilities is an open source podcast downloader application and api for the .NET Framework. Its hosted on GitHub feel free to download it or just browse the code. There is a Chocolatey install package to install the .NET Framework version (Windows) and also the Cross platform .NET Core version. There is also an Android app on the Amazon Appstore

TextByNumbers

TextByNumbers is an open source text snippet assembler for SMS and emails. Its available as an HTML5 offline application and as a native android application from Google Play or download the APK.

Worldolio

Worldolio is a shareware windows desktop, windows mobile and web application providing timezone and other geographical information. There is a Worldolio Chocolatey Package to install the desktop version.
